# Vectro 

Vectro is a modern, interactive web application built with **Next.js** and **Tailwind CSS**. It serves as a premium showcase of interactive vector illustrations, UI components, and dynamic routing features.

## ✨ Features

- **Interactive UI Components**: Includes a fully functional MacOS-like Dock for seamless navigation.
- **Dynamic Theming**: Full support for system, light, and dark modes powered by `next-themes` and Tailwind CSS.
- **Advanced Animations**: Showcases advanced SVG filtering, real-time webcam displacement mapping (Reflective Card), and smooth layout transitions.
- **Custom Error Handling**: Includes a highly customized, full-screen global `404 Not Found` page complete with SVG illustrations.
- **Responsive Layouts**: Fully responsive pages for `Archive`, `Profile`, and `Settings` demonstrating rich Tailwind CSS design patterns.
- **Modern Tech Stack**: Uses Next.js (App Router), React 19, Tailwind CSS v4, and Lucide React icons.

## 🚀 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the application running. 

## 🗺️ Navigation & Pages

- `/` - **Home**: The main Vector Gallery featuring dynamic background beams, a typing animation, and the webcam-enabled reflective security card.
- `/archive` - **Archive**: A beautiful list-view layout demonstrating file and document management interfaces.
- `/profile` - **Profile**: A sleek user profile page showcasing gradient banners, user information, and settings cards.
- `/settings` - **Settings**: A grid-based settings dashboard for managing application preferences.
- `/*` - **404 Not Found**: A custom fallback page with a beautifully integrated SVG illustration for any non-existent routes.
